# IPOS-WEB
## Getting started

```bash
# clone the project
git clone https://git.impool.com/storeros/ipos-web.git

# enter the project directory
cd ipos-web

# If the environment does not have pnpm, install pnpm first
npm install -g pnpm

# install dependency
pnpm install

# develop
pnpm  dev
```

## Build

```bash
# build for production environment
pnpm build
```
